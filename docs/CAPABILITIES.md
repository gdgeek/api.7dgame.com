## 认证与用户管理

### JWT 认证
- **用户登录** — 用户名/密码登录，返回 JWT Access Token
- **Token 刷新** — 使用 Refresh Token 获取新的 Access Token
- **Token 查询** — 查询当前用户的 Token 信息（如 access_token）

### 第三方登录
- **Apple ID 登录** — 使用 Apple ID 进行第三方认证
- **Apple ID 注册** — 使用 Apple ID 创建新账号
- **Apple ID 绑定** — 将 Apple ID 与已有账号关联
- **微信登录** — 使用微信 OpenID 登录
- **微信注册** — 使用微信账号注册新用户

### 用户信息
- **获取用户信息** — 查看当前登录用户的详细信息
- **更新用户信息** — 修改用户名称、头像等基本资料
- **创建用户档案** — 初始化用户个人档案

### 身份（Person）管理
- **身份认证** — 验证用户身份
- **创建身份** — 创建用户身份记录
- **删除身份** — 删除指定身份记录
- **身份列表** — 查看所有身份记录

## 邮箱与密码管理

### 邮箱验证
- **发送验证码** — 向用户邮箱发送验证码
- **验证邮箱** — 提交验证码完成邮箱验证
- **查询验证状态** — 查询当前邮箱验证状态

### 密码重置
- **请求密码重置** — 向已绑定邮箱发送密码重置邮件
- **验证重置 Token** — 校验密码重置令牌的有效性
- **执行密码重置** — 使用有效 Token 设置新密码

## Verse（场景）管理

Verse 是平台的核心内容单元，代表一个 AR 场景。

### 场景基本操作
- **获取场景列表** — 查询当前用户的所有场景（支持分页）
- **查看场景详情** — 获取指定场景的完整信息
- **创建场景** — 新建一个 AR 场景
- **更新场景** — 修改场景基本信息
- **删除场景** — 删除指定场景

### 场景可见性
- **设为公开** — 将场景设置为公开可见
- **取消公开** — 将场景从公开列表中移除
- **浏览公开场景** — 查看所有公开场景列表

### 场景标签
- **添加标签** — 为场景添加分类标签
- **移除标签** — 从场景中移除指定标签
- **标签管理** — 创建、查询、删除场景标签

### 场景脚本
- **查询场景脚本** — 获取场景关联的脚本内容（Blockly/Lua/JS）
- **更新场景代码** — 修改场景的脚本代码

### 场景快照
- **拍摄场景快照** — 为场景生成预览截图

---

## 场景包导入导出

支持场景数据的完整导入与导出，便于备份、迁移和分享。

- **导出场景（JSON）** — 将场景完整数据导出为 JSON 格式
- **导出场景（ZIP）** — 将场景及其资源打包导出为 ZIP 文件
- **导入场景（JSON）** — 从 JSON 文件还原场景数据
- **导入场景（ZIP）** — 从 ZIP 压缩包导入场景及资源

## 资源管理

### Meta（元数据）管理
- **Meta 列表** — 查询所有 Meta 资源
- **查看 Meta** — 获取指定 Meta 的详情
- **创建 Meta** — 新建 Meta 资源记录
- **更新 Meta** — 修改 Meta 资源信息
- **更新 Meta 代码** — 更新 Meta 关联的代码内容
- **删除 Meta** — 删除指定 Meta 资源

### Prefab（预制体）管理
- **Prefab 列表** — 查询所有 Prefab 资源
- **查看 Prefab** — 获取指定 Prefab 的详情
- **创建 Prefab** — 新建 Prefab 资源
- **更新 Prefab** — 修改 Prefab 信息
- **删除 Prefab** — 删除指定 Prefab

### Resource（资源）管理
- **资源列表** — 查询资源列表（支持分页和过滤）
- **查看资源** — 获取指定资源的详情

### 文件管理
- **文件列表** — 查询已上传文件列表
- **上传文件** — 上传普通文件
- **上传创作文件** — 上传与创作相关的文件资源

### 照片类型
- **照片类型列表** — 查询所有照片类型
- **按类型查询** — 根据类型名称筛选照片

## 教育管理

提供完整的教育机构管理能力，支持学校、班级、教师、学生的层级管理。

### 学校管理
- **查询校长信息** — 获取学校负责人信息
- **学校 CRUD** — 创建、查看、更新、删除学校信息

### 班级管理
- **班级列表** — 查询所有班级
- **按学生查询** — 查询指定学生所在班级
- **按教师查询** — 查询指定教师管理的班级
- **分配教师** — 为班级指定教师
- **移除教师** — 从班级中移除教师
- **查看我的班级** — 教师查询自己负责的班级
- **班级创建群组** — 为班级创建学习群组
- **查询班级群组** — 查看班级关联的群组

### 学生管理
- **加入班级** — 学生加入指定班级
- **查询我的信息** — 学生查看自己的班级信息

### 教师管理
- **查询我的信息** — 教师查看自己的信息

## 群组系统

群组是内容共享和协作的核心单元。

- **加入群组** — 用户加入指定群组
- **退出群组** — 用户离开指定群组
- **在群组中创建场景** — 在指定群组内新建 Verse 场景
- **删除群组场景** — 删除群组中的指定场景
- **查询群组场景** — 查看群组内所有场景列表

## API 文档

- **Swagger UI** — 可视化 API 文档界面，支持在线调试
- **OpenAPI JSON Schema** — 获取符合 OpenAPI 3.0 规范的 JSON 文档

访问地址：`http://your-domain/swagger`（需要 HTTP Basic Auth 认证）
