# Daily Questions Log

## 记录规则
- 日期格式：`YYYY-MM-DD`
- 时间格式：`HH:MM`（本地时间）
- 每条记录包含：问题、简要结论、状态
- 状态约定：`answered`、`pending`、`follow-up`
